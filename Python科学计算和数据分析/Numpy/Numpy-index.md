# Numpy 简介

<hr>

<font color=#888>Numpy(Numerical Python) 是python科学计算、数据分析以及人工智能的基础库。

<font color=#888>Numpy的核心功能是建立在一个N维数组对象**ndarray**数据结构，以及强大的**通用广播函数**之上。支持多维数组与矩阵运算，并提供了大量函数库，运行速度非常快。并提供了常用数学运算，线性代数，随机数生成傅里叶变换等函数库。
