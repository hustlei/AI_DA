# Seaborn简介

<font color=#888>Seaborn是基于matplotlib的**2D绘图库**。它在**统计**绘图方面更加方便易用，并且有自己预定义的样式。

<font color=#888>Seaborn并非用于替代matplotlib，而是对于数据分析等工作更加方便。
